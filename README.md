## Quick Start
1. npm create vite@latest examhub -- --template react-ts
2. cd examhub
3. npm install recharts @tanstack/react-router
4. Copy src\pages\* into your project src\pages\
5. Wire routes in TanStack Router
6. Connect Supabase Auth + DB
7. Set VITE_ANTHROPIC_KEY in .env for Phase 5 AI Tutor
